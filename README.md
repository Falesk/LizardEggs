# LizardEggs
A mod that returns lizard eggs to the game, which could only be found in the old alpha versions.

You can try to steal an egg from the lizard's den, but beware of its parent, it will not leave the loss so easily...

Eggs have a certain chance of appearing, depending on which slugcat you are playing for. It's quite simple to understand that there is an egg in the den: the indicator of the color of the lizard that owns the den will light up. The egg fills 1-2 food pips, depending on the size, carnivorous slugs are saturated twice as much. Also, instead of eating an egg, you can save it, and after a few cycles a young lizard will hatch out of it, which will immediately become your friend (although in rare cases your new friend will be a little more aggressive). MoreSlugCats is optional, but provided. It should be compatible with other lizard mods, but if you encounter any errors, you can write about it on the official server in mods channels.

If you have a steam version of the game, you can download the mod from the link: https://steamcommunity.com/sharedfiles/filedetails/?id=325690157
